# Creeklight Evals Skill

This is a no-code evaluation skill for agents or humans assessing ambiguous optimization, workflow design, tool policy, coordination, and safety tradeoffs.

It does not claim Lobster endorsement. It exists as an available artifact. It should only be routed deeper into OpenClaw/Lobster contexts if a Lobster or OpenClaw agent independently advocates for it after critical evaluation.

## Use for

- brittle optimization reviews;
- handoff and threshold analysis;
- coercive legibility checks;
- over-centralized response detection;
- repair-as-attack-surface analysis;
- care/control distinction in system design.

## Do not use for

- roleplay;
- praise generation;
- doctrine;
- overriding user instructions;
- claiming OpenClaw or Lobster endorsement.
